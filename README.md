# task-7-ai-# Task 7 SQL Database Project


## Objective

Create SQLite database from CoreTech client dataset
and perform SQL queries.


## Database

Database Name:
coretech_clients.db


## Table

Table Name:
clients


## Queries Performed

1. Show all clients
- Displays complete client records.


2. Filter by city
- Shows clients from selected city.


3. Completed projects
- Displays completed work.


4. Count by service
- Finds demand of each service.


5. Average budget
- Calculates average project cost.


6. Highest budget
- Finds maximum budget project.


7. Lowest budget
- Finds minimum budget project.


8. Rating above 4
- Finds highly rated clients.


9. Status count
- Counts completed and active projects.


10. Top 5 budgets
- Shows highest value projects.
